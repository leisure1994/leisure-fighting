# 核战争城市自救地图项目

## 项目概述
基于开源地图数据（OpenStreetMap + city-roads），构建中国主要城市的人防工程/地下避难所可视化地图，配套核战争生存指南。

## 技术栈
- **地图渲染**: Leaflet.js + OpenStreetMap
- **道路可视化**: 借鉴 city-roads 项目的 SVG 渲染方案
- **数据层**: 高德地图 API + 自定义人防工程数据
- **部署**: 静态网站 → a-art.top/nuclear-map

## 数据源
1. OpenStreetMap (开源地图基础数据)
2. 高德地图 API (POI搜索、地点详情)
3. 人工收集的公开人防工程信息

## 核心功能
1. 中国城市选择器
2. 地下避难所/人防工程标记
3. 道路网络可视化
4. 核战争生存指南 (侧边面板)
5. 物资清单 & 技能教程

## 中国城市列表 (首批20个)
北京、上海、广州、深圳、成都、重庆、杭州、武汉、西安、南京
天津、苏州、长沙、郑州、沈阳、青岛、宁波、东莞、无锡、佛山

## 人防工程数据字段
- 名称
- 类型 (地铁站/地下商场/人防工事/地下车库)
- 地址
- 坐标
- 容纳人数 (估算)
- 防护等级 (如有公开信息)
- 高德地图可查名称
