所有数据均来自以下公开渠道：

   - 城市建设史料

## 🗺️ 重点工程详细介绍

### 1. 北京地下城（前门东区早期人防工程）
- **地址**: 北京市东城区前门西兴隆街110号
- **规模**: 全长30余公里，最深处达十余米，可容纳30余万人
- **历史**: 始建于1969年，历时十年建成
- **特色**: 亚洲最大的早期人防工程之一，设有2300个通风孔、电影院、理发室、战备医院
- **现状**: 口部已改造为微展厅，主体尚未对外开放

### 2. 重庆建川博物馆
- **地址**: 重庆市九龙坡区谢家湾付家沟片区
- **规模**: 由24个防空洞打造，11个主题博物馆
- **特色**: 全国首个洞穴抗战博物馆聚落，展陈文物12万余件
- **历史**: 曾是抗战时期大后方的"兵器库"所在地
- **级别**: AAAA级景区

### 3. 成都一号防空洞
- **地址**: 成都市青羊区人民公园内
- **历史**: 始建于1965年，成都最早的公共人防工程
- **规模**: 建筑面积2000平方米，长200米，可容纳400人
- **温度**: 恒温25℃
- **特色**: 成都市目前唯一向市民开放的人防工事纳凉点

### 4. 武汉蛇山防空洞
- **地址**: 武汉市武昌区民主路284号/后长街76号
- **特色**: 上世纪70年代响应"深挖洞、广积粮"号召建造
- **温度**: 恒温20℃
- **规模**: 共有16个入口，可容纳300-400人
- **位置**: 距黄鹤楼步行仅550米

### 5. 杭州四牌楼纳凉点
- **地址**: 杭州市上城区四牌楼38号
- **规模**: 长约500米，总面积逾1000平方米，可容纳300多人
- **特色**: 杭州市国防动员（人民防空）体验馆
- **体验**: 可体验手摇防空警报器

## 🛠️ 前端使用示例

### 方式1: 直接引用JavaScript文件

```html
<script src="data/cities.js"></script>
<script src="data/shelters.js"></script>
<script>
  // 获取所有城市
  console.log(NuclearMapCities);
  
  // 获取所有避难所
  console.log(NuclearMapShelters);
  
  // 筛选重庆的避难所
  const chongqingShelters = NuclearMapShelters.filter(s => s.city === 'chongqing');
</script>
```

### 方式2: 加载GeoJSON到地图

```javascript
// 使用Leaflet加载
fetch('data/shelters.geojson')
  .then(response => response.json())
  .then(data => {
    L.geoJSON(data, {
      onEachFeature: function(feature, layer) {
        layer.bindPopup(`
          <h3>${feature.properties.name}</h3>
          <p>${feature.properties.address}</p>
          <p>容纳人数: ${feature.properties.capacity}</p>
        `);
      }
    }).addTo(map);
  });
```

### 方式3: 按需加载城市数据

```javascript
async function loadCityData(city) {
  const response = await fetch(`data/${city}.json`);
  const shelters = await response.json();
  return shelters;
}

// 加载重庆数据
const chongqingData = await loadCityData('chongqing');
```
