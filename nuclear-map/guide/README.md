## 快速参考卡片

### 核爆瞬间（闪光时刻）
1. **立即闭眼** - 保护视力
2. **背对闪光** - 减少皮肤和眼睛暴露
3. **就地卧倒** - 脸朝下，双手抱头
4. **等待冲击波** - 约30-60秒后冲击波到达

### 黄金15分钟行动
- **0-2分钟**：卧倒避险，检查伤势
- **2-5分钟**：向最近坚固建筑转移
- **5-10分钟**：向地下室或地下转移
- **10-15分钟**：封闭避难所，准备长期停留

### 辐射防护三原则
1. **时间** - 减少暴露时间
2. **距离** - 远离辐射源
3. **屏蔽** - 使用材料阻挡辐射

### 辐射危险等级
| 剂量率 | 等级 | 行动 |
|--------|------|------|
| <0.5 μSv/h | 安全 | 正常活动 |
| 0.5-10 μSv/h | 中等 | 减少停留时间 |
| 10-100 μSv/h | 危险 | 尽快撤离，穿戴防护 |
| >100 μSv/h | 致命 | 绝对禁区 |

### 30天最低储备（单人）
- **饮用水**：180升（6升/天）
- **食品**：约50000 kcal热量
- **医疗**：完整急救包 + 常用药品
- **防护**：N95口罩100只、防护服、盖革计数器
