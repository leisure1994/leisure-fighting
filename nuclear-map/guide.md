## 核爆炸基础知识

### 核爆炸的三重杀伤

1. **光辐射（热辐射）**
   - 爆炸瞬间产生，温度可达数千万度
   - 可造成皮肤灼伤、失明、火灾
   - 持续时间：数秒

2. **冲击波**
   - 以超音速向外传播
   - 可摧毁建筑物、造成钝器伤
   - 持续时间：数秒到数十秒

3. **核辐射**
   - 早期核辐射：爆炸后1分钟内
   - 剩余核辐射：放射性沉降，持续数天到数年
   - 最危险的长期威胁

### 杀伤半径参考（以20万吨当量为参考）

| 破坏等级 | 距离 | 特征 |
|---------|------|------|
| 极重度 | 0-1.2km | 建筑物完全摧毁，死亡率极高 |
| 重度 | 1.2-2.5km | 建筑物严重破坏，大量伤亡 |
| 中度 | 2.5-4.5km | 建筑物损坏，部分伤亡 |
| 轻度 | 4.5-7km | 玻璃破碎，轻微损伤 |

## 预警信号与响应

### 预警信号类型

1. **空袭警报**
   - 声音：鸣36秒，停24秒，反复3遍
   - 含义：敌机或导弹可能来袭
   - 行动：立即进入避难所

2. **紧急警报**
   - 声音：连续短鸣
   - 含义：核袭击即将发生或已经发生
   - 行动：就地防护或紧急撤离

3. **解除警报**
   - 声音：连续长鸣3分钟
   - 含义：危险已过或暂时安全
   - 行动：等待官方指示

### 黄金响应时间线

```
T-0: 看到闪光或听到警报
   ↓ 立即行动（0-2秒）
T+2: 趴下、掩护、闭眼、捂耳
   ↓
T+10: 冲击波到达（距离爆心3km处）
   ↓
T+30: 冲击波结束，评估环境
   ↓
T+60: 寻找/进入坚固掩体
   ↓
T+5min: 远离窗户，封堵缝隙
   ↓
T+15min: 收听官方广播
```

### 即时防护动作（"卧倒掩护"）

1. **立即趴下**
   - 面朝地面，头部朝向爆炸方向
   - 减少身体暴露面积

2. **保护头部**
   - 双手抱头，手指张开护耳
   - 嘴巴微张，减轻冲击波对耳膜的压力

3. **闭眼**
   - 避免直视闪光，防止失明
   - 即使闭眼也可能看到强光

4. **寻找掩护**
   - 利用地形、建筑物、车辆遮挡
   - 避开窗户、玻璃幕墙

## 避难所选择与改造

### 理想避难所标准

| 条件 | 要求 |
|------|------|
| 深度 | 地下至少3米，或地上建筑中间层 |
| 结构 | 钢筋混凝土，无窗户 |
| 密封 | 可封闭通风口，防止放射性粉尘 |
| 储备 | 至少14天物资 |
| 通讯 | 可接收广播信息 |

### 不同类型建筑的防护能力

**优秀**
- 地铁站（深埋站台）
- 人防工程/防空洞
- 地下车库（多层）
- 地下商场

**良好**
- 高层建筑地下室
- 银行金库
- 地铁站厅层
- 隧道

**一般**
- 普通住宅地下室
- 低层建筑内部
- 有掩体的壕沟

**避免**
- 木结构建筑
- 玻璃幕墙建筑
- 车辆内
- 开阔地带

### 简易避难所改造

#### 家庭地下室改造

1. **加固入口**
   - 厚重门板（至少5cm实木或钢板）
   - 密封条（防止粉尘进入）

2. **储备物资分区**
   - 水：每人每天4升×14天
   - 食物：非易腐食品
   - 医疗：急救包、药品
   - 工具：手摇收音机、手电筒

3. **通讯设备**
   - 电池供电收音机
   - 对讲机
   - 哨子（发出求救信号）

4. **卫生设施**
   - 便携式马桶
   - 垃圾袋
   - 消毒用品

## 生存技能

### 水净化

#### 沉淀法（最简单）
1. 静置24小时让杂质沉淀
2. 小心舀取上层清水
3. 煮沸1分钟杀菌

#### 过滤法
1. 用布、沙子、活性炭制作简易过滤器
2. 层层过滤去除杂质
3. 煮沸消毒

#### 化学消毒
- 碘片：每升水2片，静置30分钟
- 漂白剂：每升水2滴（5.25%次氯酸钠），静置30分钟
- 净水片：按说明使用

### 食物获取

#### 室内种植（长期避难）
- 豆芽：3-5天可收获
- 蘑菇：阴暗环境可培养
- 水培蔬菜：需要光源

#### 安全觅食
- 避开明显污染区域
- 检查动植物是否有辐射病症状
- 优先选择罐头、密封包装食品
- 所有野生食物需彻底清洗、去皮

### 生火技能

#### 材料准备
- 引火物：干草、纸、棉花
- 细柴：小树枝、木棍
- 粗柴：大木块

#### 方法
- 打火机/火柴（最简单）
- 放大镜聚光（需阳光）
- 摩擦取火（弓钻法、手钻法）
- 电池+钢丝棉（快速）

## 辐射防护

### 辐射类型

| 类型 | 穿透力 | 防护方式 |
|------|--------|----------|
| α射线 | 弱（一张纸阻挡） | 避免吸入/食入 |
| β射线 | 中等（几毫米铝） | 距离+屏蔽 |
| γ射线 | 强（需厚混凝土） | 距离+时间+屏蔽 |
| 中子射线 | 极强（需含氢材料） | 水、石蜡屏蔽 |

### 防护三原则

1. **时间**：减少暴露时间
   - 放射性随时间衰减（半衰期）
   - 7-10-7法则：7小时后辐射减90%，48小时后减99%

2. **距离**：远离放射源
   - 距离加倍，辐射减为1/4
   - 尽量待在地下深处

3. **屏蔽**：利用物质阻挡
   - 混凝土：60cm可阻挡大部分γ射线
   - 土壤：1米深等效于60cm混凝土
   - 水：70cm等效于60cm混凝土

### 去污程序

#### 人员去污
1. 脱去外层衣物（密封丢弃）
2. 彻底淋浴，用力搓洗
3. 剪短指甲，清洁耳道
4. 更换干净衣物

#### 物品去污
1. 刷去表面灰尘
2. 用水冲洗
3. 无法彻底去污的物品密封存放

## 医疗急救

### 辐射病症状

#### 急性辐射综合征（ARS）

**轻度（1-2 Gy）**
- 恶心、呕吐（数小时后）
- 疲劳
- 可自愈

**中度（2-4 Gy）**
- 呕吐、腹泻
- 发热
- 脱发
- 需要医疗干预

**重度（4-6 Gy）**
- 严重呕吐、腹泻
- 出血
- 感染风险极高
- 致死率50%

**极重度（>6 Gy）**
- 立即症状
- 中枢神经系统损伤
- 极高致死率

### 常见伤害处理

#### 烧伤
1. 冷水冲洗15-20分钟
2. 覆盖无菌纱布
3. 不要弄破水疱
4. 严重烧伤立即就医

#### 冲击波伤
- 骨折：固定受伤部位
- 内出血：平躺，抬高下肢
- 耳膜损伤：不要擤鼻涕

#### 切割伤
1. 直接压迫止血
2. 抬高伤肢
3. 清洁伤口，覆盖敷料
4. 深伤口需缝合

### 药品使用指南

| 症状 | 药品 | 用法 |
|------|------|------|
| 疼痛 | 布洛芬/对乙酰氨基酚 | 按说明书 |
| 腹泻 | 洛哌丁胺 | 首次4mg，后每腹泻一次2mg |
| 便秘 | 乳果糖 | 15-30ml/天 |
| 过敏 | 氯雷他定 | 10mg/天 |
| 感染 | 抗生素（需医嘱） | - |

## 心理建设

### 预期反应

**正常应激反应**
- 恐惧、焦虑
- 失眠
- 易怒
- 注意力不集中
- 这些都是正常的

### 应对策略

#### 个人层面
1. **建立规律**
   - 固定作息时间
   - 分配每日任务
   - 保持个人卫生

2. **信息管理**
   - 定时收听广播
   - 避免谣言
   - 关注官方信息

3. **身体活动**
   - 适度运动
   - 深呼吸放松
   - 保持体力

#### 群体层面
1. **分工合作**
   - 明确责任
   - 互相支持
   - 共同决策

2. **社交互动**
   - 聊天、游戏
   - 分享故事
   - 保持幽默感

3. **儿童关怀**
   - 给予安全感
   - 诚实但不过度惊吓
   - 保持日常活动
