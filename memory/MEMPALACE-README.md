# Mempalace 系统运行规则

> 此目录为 OpenClaw 与用户对话记忆的 Mempalace 空间化记忆系统。

## 宫殿结构

- `mempalace-world.json` — 全局宫殿：关系里程碑、系统规则、重要历史
- `mempalace-user-*.json` — 用户宫殿：每位用户的足迹、契约、偏好、恩怨
- `mempalace-agent.json` — 代理宫殿（我自身）：经验教训、风格调整、待办

## 房间定义

| 房间 | 含义 |
|------|------|
| `honor_hall` | 恩情、赞赏、协作成功、高光时刻 |
| `shadow_room` | 冲突、不满、失败、教训 |
| `contract_hall` | 待办、承诺、进行中的项目 |
| `footprint` | 重大行为、决策、事件 |
| `common_sense` | 稳定偏好、已知事实、风格约束 |
| `tendency_vector` | 行为倾向量化（可选） |

## 运营规则

1. **每日对话结束时**，自动提取当日关键事件，追加到对应宫殿。
2. **用户明确要求"记住"时**，立即提取并写入。
3. **权重初始值**：普通事件 3~5，重要决策 7~9，里程碑 10。
4. **硬锁事实**（`hard_lock` 标签）：永不衰减，如用户姓名、核心偏好。
5. **时间衰减**：30 天减半，常识/硬锁不衰减。
6. **Prompt 压缩**：需要调用时按 `weight × TimeDecay` 取 Top-N。
