# 核战争城市自救地图

中国主要城市地下人防工程及避难所可视化地图

## 访问地址

部署后可通过以下链接访问：
- https://a-art.top/nuclear-map/

## 数据来源

- 各地人防办公开数据
- 地铁规划资料
- 地下空间规划文件

## 覆盖城市

共27个中国主要城市，包括所有省会城市及重点城市：
北京、上海、广州、深圳、成都、武汉、西安、重庆、南京、杭州、天津、青岛、大连、厦门、苏州、长沙、郑州、沈阳、哈尔滨、昆明、合肥、南昌、贵阳、福州、南宁、兰州、乌鲁木齐

## 技术栈

- Leaflet.js 地图引擎
- CartoDB Dark Matter 地图底图
- 纯前端实现，无需后端
