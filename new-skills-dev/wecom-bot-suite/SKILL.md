## 快速开始

### 1. 配置企业微信

```bash
cp config.example.json config.json
# 填入企业微信Webhook密钥和配置
```

### 2. 配置自动回复规则

```json
{
  "auto_reply": [
    {
      "keyword": "价格",
      "response": "我们的产品价格请查看价目表：xxx"
    },
    {
      "keyword": "帮助",
      "response": "可用指令：价格|地址|联系方式"
    }
  ]
}
```

### 3. 启动机器人

```bash
./wecom-bot.sh start
```
