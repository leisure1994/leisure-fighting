```javascript

```

## 避难所类型分布

- **火车站地下避难所**：每个城市1-2个，基于主要铁路枢纽
- **高铁站地下避难所**：高铁沿线城市配备
- **地铁站**：拥有地铁系统的城市
- **人防地下商场/商城**：市中心商业区
- **工业企业地下指挥所**：重点工业城市配备（核5级）
- **港口地下指挥中心**：沿海/沿江港口城市
- **煤矿/油田地下防空洞**：资源型城市依托矿井建设
- **机场地下避难所**：重要交通枢纽
- **水利设施防护工程**：水电站周边
- **航天/科研基地防护**：特殊重要设施

## 数据特点

1. **真实坐标**：基于高德/百度地图真实城市坐标
2. **合理分布**：每个城市3-5个避难所，覆盖主要区域
3. **分级防护**：
   - 核5级：重要工业/能源/军事设施
   - 核6级：一般民用防护工程
4. **潜在目标标注**：识别可能优先受打击的目标
5. **容量估算**：基于工程规模估算合理容纳人数
